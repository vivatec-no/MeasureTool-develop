MeasureTool Develop Builds

This repository stores compiled build artifacts for the MeasureTool project.

It does not contain source code.

Artifacts are automatically uploaded by GitHub Actions from the main MeasureTool repository.

Contents of releases may include:

Python wheels (.whl)

Python source distributions (.tar.gz)

Frontend build archives (react-ui-dist.zip)

These files are intended for deployment and device updates (e.g., Raspberry Pi installations).
